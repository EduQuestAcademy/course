Lesson 7.1

Understanding Multiplication: Equal Groups and Arrays

Multiplication is just a fast way to count equal groups!

What Are We Learning Today?

Multiplication is one of the most powerful tools in all of mathematics. Instead of adding the same number over and over, multiplication lets you find the total of equal groups in one quick step. Today we learn what multiplication really means using equal groups and arrays!

Key Words to Know

Multiplication -- A fast way to add equal groups together.
Factor -- The numbers being multiplied together. In 4 x 3, both 4 and 3 are factors.
Product -- The answer to a multiplication problem. In 4 x 3 = 12, the product is 12.
Equal Groups -- Groups that all have the same number of items.
Array -- Objects or symbols arranged in rows and columns to show multiplication.
Skip Counting -- Counting by a number other than 1 (by 2s, 3s, 5s, etc.).

Equal Groups -- The Big Idea of Multiplication

The Long Way (Adding)

3 bags, each with 4 apples:

4 + 4 + 4 = 12

This works, but it takes a long time for big numbers!

The Fast Way (Multiplying)

3 groups of 4:

3 x 4 = 12

Same answer -- much faster! Multiplication IS repeated addition.

Reading Multiplication Equations

3 x 4 = 12
3 groups    x    4 in each group    =    12 total

We read 3 x 4 as "3 times 4" or "3 groups of 4."

EquationRead asMeans
2 x 5 = 102 times 52 groups of 5 = 10 total
4 x 3 = 124 times 34 groups of 3 = 12 total
5 x 6 = 305 times 65 groups of 6 = 30 total

Arrays -- Rows and Columns

An array is a neat arrangement of objects in rows (across) and columns (down). Arrays make it easy to see multiplication!

3 rows x 4 columns = 12

3 x 4 = 12   OR   4 x 3 = 12 (same array, different view!)

Notice: You can read the same array two ways! 3 rows of 4 = 12 AND 4 columns of 3 = 12. This is called the Commutative Property -- order doesn't change the product!

Try It Yourself!

  1. Draw an array for 2 x 6. How many dots in total?
  2. Write a multiplication equation for: "5 bags with 3 oranges each."
  3. There are 4 rows of chairs with 7 chairs in each row. Write a multiplication equation and find the total.
  4. Skip count by 4s to find: 6 x 4 = ___. (4, 8, 12...)

Remember This!

Multiplication = equal groups added together quickly.

Factor x Factor = Product

Arrays show multiplication in rows and columns.

Multiplication Facts: Strategies and Fluency

Master your times tables with smart strategies!

What Are We Learning Today?

Knowing your multiplication facts quickly and accurately -- without having to stop and count -- is one of the most useful skills in all of math! Today we learn smart strategies to help you master your times tables, plus practice tips to build fluency.

Key Words to Know

Multiplication Facts -- The basic multiplication equations from 0x0 up to 10x10 (or 12x12) that you should know from memory.
Fluency -- Being able to recall facts quickly and accurately, without needing to count.
Strategy -- A smart method or trick for figuring out a fact you are not sure of.

The x9 Finger Trick!

How It Works

Hold up all 10 fingers. To find 9 x 4, fold down your 4th finger. Count fingers to the LEFT = tens digit. Count fingers to the RIGHT = ones digit.

9 x 4: fold 4th finger → 3 fingers left, 6 fingers right → 36!

The x9 Pattern

The digits in every 9x product always add up to 9!

9x1=09 (0+9=9), 9x2=18 (1+8=9), 9x3=27 (2+7=9), 9x4=36 (3+6=9), 9x5=45 (4+5=9)...

Understanding Division: Sharing and Grouping

Division is all about splitting things up fairly!

What Are We Learning Today?

Division is the operation we use when we need to split a number into equal groups. You use division every day -- sharing pizza, dividing up cards, splitting into teams. Today we learn two ways to think about division: sharing equally and making equal groups.

Key Words to Know

Division -- The operation of splitting a number into equal groups.
Dividend -- The number being divided. In 12 / 3, the dividend is 12.
Divisor -- The number you divide by. In 12 / 3, the divisor is 3.
Quotient -- The answer to a division problem. In 12 / 3 = 4, the quotient is 4.
Remainder -- The amount left over when a number cannot be divided equally.

Two Ways to Think About Division

Sharing Equally (Partitive)

You know how many groups. You need to find how many are in each group.

"12 cookies shared equally among 4 friends. How many does each friend get?"

12 / 4 = 3 each

Making Equal Groups (Quotitive)

You know how many in each group. You need to find how many groups.

"12 cookies placed 4 on each plate. How many plates are needed?"

12 / 4 = 3 plates

Reading Division Equations

12 ÷ 3 = 4
12 (dividend)   ÷   3 (divisor)   =   4 (quotient)
EquationRead asMeans
20 ÷ 4 = 520 divided by 4 equals 520 split into 4 equal groups = 5 in each group
18 ÷ 3 = 618 divided by 3 equals 618 split into 3 equal groups = 6 in each group
35 ÷ 5 = 735 divided by 5 equals 735 split into 5 equal groups = 7 in each group

Special Division Rules

Try It Yourself!

  1. Write a division equation: "24 stickers shared equally among 6 friends."
  2. Solve: 36 ÷ 4 = ___   |   45 ÷ 9 = ___   |   28 ÷ 7 = ___
  3. Draw a picture showing 15 ÷ 3 = 5 (use circles as groups and dots as objects).
  4. What is the quotient of 0 ÷ 7? What about 7 ÷ 7?

Remember This!

Dividend ÷ Divisor = Quotient

Division = splitting into equal groups (sharing OR grouping)

Any number ÷ 1 = itself  |  Any number ÷ itself = 1  |  0 ÷ any number = 0

The Relationship Between Multiplication and Division

Multiplication and division are opposites -- and knowing one helps you with the other!

What Are We Learning Today?

Multiplication and division are inverse operations -- they are mathematical opposites that undo each other, just like addition and subtraction. When you know a multiplication fact, you automatically know two division facts! This is called a fact family.

Key Words to Know

Inverse Operations -- Operations that undo each other. Multiplication and division are inverse operations.
Fact Family -- A set of related multiplication and division equations that use the same three numbers.
Related Facts -- Equations that are connected and can be used to solve each other.

Fact Families -- Three Numbers, Four Facts!

Fact Family for 3, 4, and 12

3 x 4 = 12
4 x 3 = 12
12 ÷ 3 = 4
12 ÷ 4 = 3

Fact Family for 5, 7, and 35

5 x 7 = 35
7 x 5 = 35
35 ÷ 5 = 7
35 ÷ 7 = 5

Using Multiplication to Solve Division

When you see a division problem, turn it into a missing-factor multiplication problem!

Division ProblemThink of it as...Answer

42 ÷ 7 = ___7 x ___ = 42?6 (because 7 x 6 = 42)
56 ÷ 8 = ___8 x ___ = 56?7 (because 8 x 7 = 56)
63 ÷ 9 = ___9 x ___ = 63?7 (because 9 x 7 = 63)
Strategy: Instead of struggling with a hard division fact, ask yourself: "What times the divisor equals the dividend?" Your multiplication knowledge saves the day!

Try It Yourself!

  1. Write the complete fact family for: 6, 8, and 48.
  2. Write the complete fact family for: 3, 9, and 27.
  3. Use multiplication to solve: 54 ÷ 6 = ___ (think: 6 x ___ = 54)
  4. Use multiplication to solve: 72 ÷ 8 = ___ (think: 8 x ___ = 72)

Remember This!

Multiplication and division are inverse operations -- they undo each other.

Every set of three numbers makes a fact family of 4 equations.

To solve division, ask: "What times the divisor = dividend?"

Lesson 7.5

Solving Word Problems with Multiplication and Division

Apply your skills to real-life math situations!

What Are We Learning Today?

Math facts are most powerful when you can use them to solve real-world problems! Word problems tell a story with numbers. Your job is to read carefully, figure out what operation to use, and solve it step by step. Today we learn a reliable strategy for tackling any word problem!

Key Words to Know

Word Problem -- A math problem written as a real-life story or situation.
Operation -- The mathematical action to perform: add, subtract, multiply, or divide.
Key Words -- Words in a problem that hint at which operation to use.
Unknown -- The number you are trying to find -- what the question is asking for.

Key Words That Signal Multiplication or Division

Multiply when you see...

  • each, every, per
  • groups of, rows of
  • times, twice, double
  • in all, total, altogether

Divide when you see...

  • share equally, split
  • each person gets
  • how many groups
  • cut into, separated into

The 4-Step Problem-Solving Strategy

1
UNDERSTAND -- Read the problem carefully. What do you know? What are you trying to find?
2
PLAN -- What operation will you use? Write a number sentence with a box for the unknown.
3
SOLVE -- Calculate the answer. Show your work!
4
CHECK -- Does the answer make sense? Use the inverse operation to verify.

Problem 1 (Multiplication)

A bookshelf has 6 shelves. Each shelf holds 8 books. How many books are on the bookshelf in all?

Step 1: I know 6 shelves, 8 books each. I need the total.

Step 2: "In all" + "each" = multiply. 6 x 8 = ___

Step 3: 6 x 8 = 48 books

Step 4: Check: 48 ÷ 6 = 8 ✓

Problem 2 (Division)

A teacher has 32 stickers to share equally among 8 students. How many stickers does each student get?

Step 1: I know 32 stickers total, 8 students. I need to know how many each student gets.

Step 2: "Share equally" = divide. 32 ÷ 8 = ___

Step 3: 32 ÷ 8 = 4 stickers each

Step 4: Check: 4 x 8 = 32 ✓

Try It Yourself!

  1. A garden has 7 rows of flowers with 9 flowers in each row. How many flowers are there in all?
  2. A box of 56 crayons is divided equally into 8 bags. How many crayons are in each bag?
  3. Sam earns $6 for every hour he helps at home. He worked for 5 hours. How much did he earn?
  4. Write your OWN word problem using multiplication or division. Solve it and show all four steps.

Remember This!

Each, groups of, in all = usually multiply.

Share equally, split, each person gets = usually divide.

Always: Understand → Plan → Solve → Check!

Properties of Multiplication

Powerful rules that make multiplication easier and faster!

What Are We Learning Today?

Mathematicians have discovered special rules about how multiplication works -- called properties. These properties are not just interesting facts; they are powerful tools that let you solve problems faster, check your work, and break hard problems into easier ones!

Key Words to Know

Commutative Property -- You can multiply numbers in any order and get the same product. a x b = b x a.
Associative Property -- When multiplying three numbers, you can change the grouping and get the same product. (a x b) x c = a x (b x c).
Distributive Property -- You can break a factor into parts, multiply each part, then add the results. a x (b + c) = (a x b) + (a x c).
Identity Property -- Any number multiplied by 1 equals itself. a x 1 = a.
Zero Property -- Any number multiplied by 0 equals 0. a x 0 = 0.

The Five Properties -- Explained Simply

1. Commutative Property -- "Order doesn't matter!"

3 x 5 = 15    AND    5 x 3 = 15

This cuts the number of facts you need to memorise nearly in HALF! If you know 3 x 7, you already know 7 x 3.

2. Associative Property -- "Grouping doesn't matter!"

(2 x 3) x 4 = 6 x 4 = 24    AND    2 x (3 x 4) = 2 x 12 = 24

When you multiply three numbers, start with whichever pair is easiest!

3. Distributive Property -- "Break it apart!"

6 x 7 = 6 x (5 + 2) = (6 x 5) + (6 x 2) = 30 + 12 = 42

Break a tricky fact into easier parts. Perfect for facts you haven't memorised yet!

4. Identity Property -- "Times 1 = itself!"

9 x 1 = 9      1 x 47 = 47      1 x 1,000 = 1,000

Multiplying by 1 changes nothing -- the number keeps its identity!

5. Zero Property -- "Times 0 = always 0!"

9 x 0 = 0      0 x 500 = 0      0 x 0 = 0

No matter how large the number, multiplying by zero always gives zero.

Using the Distributive Property to Solve Harder Facts

Hard FactBreak It ApartSolve Each PartAdd
7 x 87 x (4 + 4)(7 x 4) + (7 x 4)28 + 28 = 56
6 x 96 x (10 - 1)(6 x 10) - (6 x 1)60 - 6 = 54
8 x 78 x (5 + 2)(8 x 5) + (8 x 2)40 + 16 = 56

  2. Use the Distributive Property to solve 7 x 6 by breaking it into 7 x (5 + 1).
  3. Use the Associative Property: (3 x 2) x 5 = 3 x (2 x 5). Solve both ways -- do you get the same answer?
  4. What is 1,000 x 0? What property tells you the answer instantly?
